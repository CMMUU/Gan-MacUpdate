# UpdateMate

[English](./README.en.md)

> 你的 Mac，你决定是否现在更新。
>
> 一个基于 hosts 的 macOS 一键拦截更新工具。

## 这是什么？

UpdateMate 是一个基于 `hosts` 的小工具，用来一键拦截 macOS 系统更新相关请求，减少更新提示和更新打扰。

不讲复杂故事，不聊宏大路线。

它现在的核心意思就一句话：

**一键拦截，图个清净。**

## 功能

- 一键写入 hosts 规则
- 一键恢复默认状态
- 减少系统更新相关打扰
- 适合想暂时不被更新提醒影响的用户

## 仓库结构

- `src/` + `src-tauri/`：当前桌面端原型
- `noupdate-server/`：相关服务端实验代码
- `deploy/`：部署配置样例
- `docs/`：补充文档
