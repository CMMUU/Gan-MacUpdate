UpdateMate Docs

UpdateMate is designed to help users decide when their Mac updates, using a calmer and more compliant “update schedule control” narrative.

Core pricing

  • Global: Standard $9.99 / version, Pro $19.99 / version
  • Mainland China: ¥68 / version, ¥128 / version
  • Free trial: 7 days full access

Strategies

  • Stability First
  • Version Lock
  • Full Control

Technology direction

The long-term client direction is SwiftUI + AppKit, while licensing and activation are planned around Spring Boot and SQLite. The current web prototype remains a transition layer.
